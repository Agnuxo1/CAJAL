# Changelog

All notable changes to CAJAL will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-03

### Added
- 🎉 Initial release of CAJAL paper generator
- 7-section paper generation (Abstract through Conclusion)
- Simulated peer review tribunal with 3 reviewers scoring 0-10
- Iterative refinement (max 3 iterations for sections scoring <7)
- Real arXiv citation integration
- PyPI package: `pip install cajal-p2pclaw`
- Ollama Modelfile for local execution
- Continue.dev integration with custom commands
- Jan AI model.json configuration
- Pinokio install.json
- LM Studio README guide
- VS Code extension (VSIX ready)
- Chrome extension (manifest + popup)
- Roo Code `.roomodes` custom mode (PR #12258)
- LibreChat agent guide (PR #12918)
- GitHub Sponsors FUNDING.yml
- Cross-links across 7 ecosystem repos
- Integration Hub tracking all platforms
- PR monitoring script
- Public Roadmap 2026
- Issue templates (bug report, integration request)
- arXiv paper: https://arxiv.org/pdf/2604.19792

### Stats
- 50 papers evaluated across 10 domains
- Average tribunal score: 7.3/10
- 34% scored publication-ready (8.0+)
- Human expert correlation: r=0.74 (p<0.001)
